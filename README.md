Sammanfattning av mitt EDA-projekt: Bostadsmarknaden i Kalifornien
I det här projektet har jag gjort en så kallad Exploratory Data Analysis (EDA) på ett dataset om bostäder i Kalifornien. Som nybörjare i Pythonprogrammering har målet varit att gå från att bara "titta på siffror" till att faktiskt förstå vad som påverkar priset på ett hem.
